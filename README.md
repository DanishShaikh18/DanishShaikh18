## Projects

### Multi-Agent RAG System
A document Q&A system built with LangGraph, Gemini, and FAISS. Runs queries through a pipeline of agents — guardrail, research, retrieval, response, and validation — before returning a grounded answer with source references. Built a Streamlit UI on top to make it usable.

**Stack:** Python · LangGraph · LangChain · Gemini · FAISS · Streamlit

---

### LLM Evaluation Pipeline
An evaluation framework for RAG-generated answers using an LLM-as-a-judge approach. Takes a benchmark dataset of question-answer pairs, runs them through a RAG pipeline, and scores responses on correctness, completeness, relevance, and hallucination risk. Outputs structured JSON reports.

**Stack:** Python · LangChain · Gemini · FAISS · Sentence Transformers · Pydantic
